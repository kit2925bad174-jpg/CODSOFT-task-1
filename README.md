PERSONAL PORTFOLIO WEBSITE

A responsive personal portfolio website created using HTML and CSS. This project showcases personal information, skills, projects, resume, and contact details in a clean and professional design.

FEATURES

• Header section with name and introduction
• About section with profile image and bio
• Skills section displaying technical skills
• Projects section with project details and images
• Resume download option
• Contact information section
• Footer with copyright information
• Responsive and attractive layout using CSS

TECHNOLOGIES USED

• HTML5
• CSS3

SECTIONS INCLUDED

• Header
• About Me
• Skills
• Projects
• Resume
• Contact
• Footer
